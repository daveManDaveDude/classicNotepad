# clasicNotepadCrossPlatform (planning docs bootstrap)

This directory contains the initial research and planning pack for building a native cross-platform successor to Classic Notepad.

Current contents:
- `docs/01_TECHNOLOGY_RESEARCH_AND_DECISION.md`
- `docs/02_CURRENT_CLASSIC_NOTEPAD_BEHAVIOR_SPEC.md`
- `docs/03_PLATFORM_API_MAPPING.md`
- `docs/04_IMPLEMENTATION_PLAN_MAC_FIRST.md`

Notes:
- Dark mode is explicitly out of v1 scope.
- Plan is macOS-first implementation, then Linux parity, while keeping buildability on macOS/Linux/Windows.
